# Reviewer Personas

This directory defines reusable reviewer personas for spec review, plan review, implementation review, and final quality gates.

## Personas
- `code-reviewer.md`: practical engineering reviewer focused on implementation quality and best practices
- `ui-ux-reviewer.md`: usability and interface reviewer focused on Fluent UI consistency, accessibility, and workflow clarity
- `implementation-reviewer.md`: execution reviewer focused on delivery realism, dependency sequencing, and implementation readiness

## Usage
Use these personas when reviewing:
- product and feature specs
- contracts and OpenAPI artifacts
- implementation plans and tasks
- UI mockups and implemented screens
