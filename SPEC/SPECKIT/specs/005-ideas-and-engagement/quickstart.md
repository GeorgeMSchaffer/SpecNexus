# Quickstart: Ideas and Engagement

1. Create an idea without specifying status and verify it lands in the left-most swimlane.
2. Add tags with mixed casing and whitespace and verify normalization.
3. Mention a same-organization user by email in an idea and in a comment.
4. Verify Read Only users can comment and upvote but cannot edit idea content.
5. Verify a board-configured User can move any idea on the board.
6. Verify completed ideas remain editable.
